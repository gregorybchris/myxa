## Installation

Install using [uv](https://docs.astral.sh/uv)

```bash
uv sync
```

## Usage

```bash
uv run myxa
```

## Package name

Myxa is named after the slime mold _Myxogastria_, the Ancient Greek word μύξα _myxa_, meaning "mucus".
